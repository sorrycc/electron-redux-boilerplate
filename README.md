# electron-redux-boilerplate

## Get Start

Clone this repo.

```bash
$ git clone git@github.com:sorrycc/electron-redux-boilerplate.git electron-app
```

Install dependencies.

```bash
$ npm i
```

Build src to dist and watch.

```
$ npm run build
```

Open another terminal tab.

```bash
$ npm start
```
